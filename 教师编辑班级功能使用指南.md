# 📝 教师端编辑学生班级 - 使用指南

## ✨ 新功能

现在您可以**直接在教师后台界面**编辑学生的班级，无需操作数据库！

---

## 🚀 如何使用

### 第1步：进入教师后台

1. 打开应用
2. 点击右上角"教师后台"按钮
3. 进入"学生"标签页

### 第2步：找到要编辑的学生

- 使用班级筛选器快速定位
- 或在列表中滚动查找

### 第3步：点击编辑按钮

在学生姓名旁边，您会看到一个 ✏️ 编辑按钮，点击它。

### 第4步：修改班级名称

在弹出的对话框中：
1. 查看学生当前班级
2. 输入新的班级名称
3. 系统会自动显示标准化后的名称
4. 点击"保存更改"

### 第5步：确认更新

- 系统会显示更新成功的提示
- 列表会自动刷新
- 该学生的所有历史记录都已更新

---

## 🎯 功能特点

### ✅ 自动标准化

输入任何格式，系统自动转换为标准格式：

| 您输入 | 自动转为 |
|--------|---------|
| `a乙2` | `A乙2` |
| `A乙二班` | `A乙2` |
| ` A 乙 2 ` | `A乙2` |

### ✅ 批量更新

点击一次保存，自动更新：
- 该学生的所有练习记录
- 学生信息表
- 班级统计数据

### ✅ 实时预览

输入班级名称时，立即看到标准化结果。

### ✅ 安全确认

- 清晰显示影响的记录数
- 可以随时取消操作

## 📊 使用场景

### 场景1：修正错误输入

**问题：** 学生刘宇航填写了"AZ2"，但应该是"A乙2"

**解决：**
1. 找到刘宇航
2. 点击编辑按钮
3. 输入 `A乙2`
4. 保存

**结果：** 刘宇航的所有练习记录都更新为"A乙2"

### 场景2：统一班级格式

**问题：** 同一个班级有多种写法（A甲2、A甲2班、a甲2）

**解决：**
1. 逐个学生点击编辑
2. 统一输入标准格式 `A甲2`
3. 保存

**结果：** 所有学生班级名称统一

### 场景3：学生转班

**问题：** 学生从A甲2转到A乙2

**解决：**
1. 找到该学生
2. 点击编辑
3. 输入新班级 `A乙2`
4. 保存

**结果：** 学生的所有记录都归入新班级

## 💡 使用技巧

### 技巧1：批量修正

如果有多个学生需要修改：
1. 先记下需要修改的学生名单
2. 逐个点击编辑
3. 统一输入标准班级名称

### 技巧2：利用筛选器

1. 先选择"全部班级"
2. 查看所有学生
3. 找出格式不统一的
4. 逐个修正

### 技巧3：验证结果

修改后：
1. 切换到"班级"标签页
2. 查看班级统计是否正确
3. 返回"学生"标签页
4. 使用筛选器测试

---

## ⚠️ 注意事项

### 重要提示

1. **修改会影响所有历史记录**
   - 该学生的所有练习记录都会更新
   - 无法撤销，请谨慎操作

2. **保存前确认**
   - 仔细检查新的班级名称
   - 查看标准化预览

3. **避免频繁修改**
   - 尽量让学生首次填写时就正确
   - 减少后期修改的工作量

### 最佳实践

1. **统一命名规范**
   - 告知学生标准的班级命名格式
   - 例如：A甲2、B乙3

2. **定期检查**
   - 每周查看一次班级列表
   - 及时修正格式不统一的情况

3. **教育学生**
   - 提醒学生认真填写班级信息
   - 说明班级信息的重要性

## 🔧 技术说明

### 自动更新的内容

当您保存更改时，系统会：

1. **更新 practice_records 表**
   ```sql
   UPDATE practice_records
   SET class_name = '新班级'
   WHERE student_name = '学生姓名';
   ```

2. **更新 students 表**
   ```sql
   UPDATE students
   SET class_name = '新班级'
   WHERE student_name = '学生姓名';
   ```

3. **刷新视图**
   - `student_summary` 视图自动更新
   - `class_stats` 视图自动更新

4. **重新加载数据**
   - 教师后台数据自动刷新
   - 显示最新的统计结果

## ❓ 常见问题

### Q: 修改会影响学生的练习记录吗？

A: 会更新班级字段，但不会影响其他数据（成绩、时间等）。

### Q: 可以撤销修改吗？

A: 不能自动撤销，但您可以再次编辑改回原来的班级。

### Q: 修改后需要刷新页面吗？

A: 不需要。保存后数据会自动刷新。

### Q: 可以同时修改多个学生吗？

A: 目前是逐个修改。如需批量修改，请使用数据库SQL。

### Q: 标准化规则可以自定义吗？

A: 可以。修改 `classNameNormalizer.ts` 文件中的规则。
