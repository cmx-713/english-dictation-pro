# ⚡ 快速修复："班"字问题

## 🎯 问题

您发现 `A甲2` 和 `A甲2班`、`A乙2` 和 `A乙2班` 被识别为不同班级。

## ✅ 立即解决（2分钟）

### 方法1：直接在 Supabase 中修改（推荐）

1. 登录 [Supabase Dashboard](https://app.supabase.com)
2. 进入 **SQL Editor**
3. 执行以下SQL：

```sql
-- 查看当前有哪些带"班"字的班级
SELECT 
  class_name as 当前名称,
  REGEXP_REPLACE(class_name, '班$', '') as 修改后,
  COUNT(*) as 记录数
FROM practice_records 
WHERE class_name LIKE '%班'
GROUP BY class_name
ORDER BY class_name;
```

4. 确认无误后，执行修复：

```sql
-- 去除 practice_records 表中末尾的"班"字
UPDATE practice_records
SET class_name = REGEXP_REPLACE(class_name, '班$', '')
WHERE class_name LIKE '%班';

-- 去除 students 表中末尾的"班"字
UPDATE students
SET class_name = REGEXP_REPLACE(class_name, '班$', '')
WHERE class_name LIKE '%班';
```

5. 验证结果：

```sql
-- 查看班级统计
SELECT * FROM class_stats ORDER BY class_name;
```

**完成！** 现在 `A甲2班` → `A甲2`，`A乙2班` → `A乙2`

## 📝 手动修改班级名称（通用方法）

如果您需要修改任何班级名称：

```sql
-- 1. 查看当前的班级列表
SELECT DISTINCT class_name 
FROM practice_records 
ORDER BY class_name;

-- 2. 修改指定的班级名称
-- 例如：将"旧名称"改为"新名称"
UPDATE practice_records
SET class_name = '新名称'
WHERE class_name = '旧名称';

UPDATE students
SET class_name = '新名称'
WHERE class_name = '旧名称';

-- 3. 验证结果
SELECT * FROM class_stats ORDER BY class_name;
```

### 批量修改示例：

```sql
-- 将所有包含"1班"的改为"1"
UPDATE practice_records
SET class_name = REPLACE(class_name, '1班', '1')
WHERE class_name LIKE '%1班';

-- 使用正则表达式批量替换
UPDATE practice_records
SET class_name = REGEXP_REPLACE(class_name, '某个模式', '替换内容')
WHERE class_name ~ '某个模式';
```
